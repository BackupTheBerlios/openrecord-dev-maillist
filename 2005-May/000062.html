I did a little more work today on the login UI.  While I was at it I created
user accounts for all of us, in the demo repository.  

If you check out the latest release and open demo_page.html, you should be
able to log in without having to create a new account first.  I didn't make
passwords, so just leave the password field blank.  You can add a password
if you want to.  We don't yet have a UI for changing your password, but you
can edit the user list file in the repository and then check in your
changes.
