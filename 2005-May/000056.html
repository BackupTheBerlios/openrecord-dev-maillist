That's a great page.  I think we should do a number of things just like
that.

I added one new task to our to-do list, about letting the user press the
Escape key when editing.

I think it would be good if we gave the user visual cues about what fields
are editable.  Actually, our code used to do a better job of that, back in
April.  Try going to our old April demo page:
  http://openrecord.org/demo/current/trunk/source/demo_page.html

If you press the "Edit" button, then you see bordered "edit boxes" whenever
you hover the mouse over the text fields.  Our current code doesn't give
visual cues until you actually click on the text.  Is that something we
should fix before 0.1?
