Hey Chao,

Here's the API that the new data model offers for creating users/accounts
and logging in (see below).  

You can also look at the data model unit tests, to see some examples of code
that creates new users and logs them in and out:
http://svn.berlios.de/viewcvs/openrecord/trunk/source/model/ModelTest.js?vie
w=markup

The new data model already has a working implementation of this API,
although we can always refactor the API and implementation if this turns out
to be the wrong design.

Good luck!

:o) Brian
