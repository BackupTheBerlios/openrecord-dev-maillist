There is one Dojo package, dojo.storage, that does make use of flash. 
The dojo.storage package doesn't actually use any of the UI features of 
flash, but instead uses a feature that can save data to a cache on the 
user's local hard drive, and dojo.storage makes that feature available 
to dojo apps through a JavaScript API.

I don't know much about flash, or about how flash and JavaScript talk to 
each other.  It would be cool if we could use the gapminder component in 
JavaScript.  It would also be possible for someone to write a similar 
widget in pure JavaScript, given a year or two to work on it!
